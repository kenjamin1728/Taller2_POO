## Descripción del Proyecto
Este sistema simula la experiencia de un juego de pokemon, incluyendo la captura por hábitats con probabilidades, sistema de combate con tabla de tipos y persistencia de datos (guardado/carga).

## Estructura del Proyecto
* `App.java`: Clase principal con la lógica de menús y flujos.
* `Entrenador.java`: Gestión de equipo y PC del jugador.
* `Pokemon.java`: Definición de atributos y estados del Pokémon.
* `Gimnasio.java` / `AltoMando.java`: Gestión de líderes y desafíos.
* `TablaTipos.java`: Matriz de efectividad para cálculos de daño.

## Instrucciones de Ejecución
1. Clonar el repositorio.
2. Abrir Eclipse IDE.
3. Importarlo
4. Seleccionar la carpeta del repositorio y finalizar.
5. Ejecutar `App.java`.
